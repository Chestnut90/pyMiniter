# Mini Tweeter for backend study.

## written by python 3.x

## Dependencies
	*flask*
		- pip install flask
	*httpie*
		- pip install httpie

## Usage
	*Windows*
		1. open cmd
		2. run FLASK_APP using 'python run.py' in root.
		3. open other cmd
		4. communicate with 'http' command
			- ex) http [method] [http://127.0.0.1/ping]
